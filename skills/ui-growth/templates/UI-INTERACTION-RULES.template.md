# UI Interaction Rules

## 1. 交互原则

- 反馈应靠近操作发生位置；
- 用户应能判断当前状态、下一步和是否成功；
- 不可逆操作必须有与风险匹配的确认；
- Loading 不应造成重复提交或布局跳动；
- 键盘、鼠标和触控交互保持一致语义。

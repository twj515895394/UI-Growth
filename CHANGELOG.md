# Changelog

本项目使用语义化版本表达 Skill 行为变化。

## [0.1.0] - 2026-07-23

### Added

- 创建 `UI Growth` 主 Skill；
- 增加 UI 视觉问题分类体系；
- 增加用户口语与专业术语映射；
- 增加专业澄清和风险分级协议；
- 增加项目 UI 现状审计流程；
- 增加设计规范治理流程；
- 增加 UI 修改执行合同；
- 增加截图、状态、响应式和设计一致性验收流程；
- 增加项目 UI 文档模板；
- 增加卡片间距调整完整示例；
- 增加对 `ibelick/ui-skills` 的借鉴分析与边界说明；
- 增加架构、路线图和 Agent 贡献规则。

### Direction

- 面向缺少前端 UI 专业知识的开发者；
- 不绑定前端框架、组件库或视觉风格；
- 以“意图理解 → 专业确认 → 最小修改 → 视觉验收 → 规范沉淀 → 渐进学习”为核心闭环。
